# 🐾 Random Pet Generator

- 🐱 Random Cat Facts
- 🐶 Random Dog Images

## 🚀 Features
- API integration using Axios
- Async/Await handling
- Error handling
- Clean UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Axios

## 🌐 APIs Used
- https://catfact.ninja/
